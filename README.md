# Geocaching PWA - Odkrywca Skrytek

![Geocaching](https://static.vecteezy.com/system/resources/previews/067/953/199/non_2x/golden-3d-map-pin-location-pointer-isolated-on-transparent-background-png.png)

## O aplikacji
Aplikacja terenowa typu Progressive Web App (PWA) ułatwiająca zabawę w geocaching. Umożliwia zrobienie zdjęcia znalezionej skrytki, automatyczne pobranie lokalizacji GPS, zaznaczenie znaleziska na mapie OpenStreetMap oraz udostępnienie odkrycia znajomym.
